## Critical Anti-Patterns (do NOT repeat)

| Pattern | Description | Severity | Prevention |
|---------|-------------|----------|------------|
| Dropped event/IOC field on merge | A new optional `ForensicEvent`/`IOC` field silently disappears after a merge or dedup | **blocking** | Wire it through `stateTypes.ts` + `responseSchema.ts` (optional) + `stateMerge.ts` **both** branches (update + push) + `correlate.ts` `mergeGroup`. New importer must populate it + tag `sources`. |
| NUL bytes in files | Some Write/Edit ops injected `\x00` (breaks later Edits with "string not found") | advisory | If an Edit won't match, `sed -n` / `cat -v` for `^@`; strip with `perl -i -pe 's/\x00//g'`. |
| Truncated AI JSON | `max_tokens` too low cuts a big synthesis mid-array | advisory | `parseJsonLoose` repairs it and the high-severity backfill covers omitted findings; raise `DFIR_AI_MAX_TOKENS` if many findings drop. Too high re-risks OpenRouter 402. |
